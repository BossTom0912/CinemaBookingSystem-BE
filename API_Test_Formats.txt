CinemaSystem API test formats
Base URL Swagger: https://localhost:7122

Seeded login accounts:
- Customer: customer@test.com / Test@123
- Staff:    staff@test.com    / Test@123
- Manager:  manager@test.com  / Test@123
- Admin:    admin@test.com    / Test@123

Seeded IDs:
- cinemaId: CIN_TEST_ALL_ROLES
- roomId: ROOM_TEST_A
- movieId: MOV_TEST_ALL_ROLES
- showtimeId: SHOW_TEST_20260610_1900
- seatId: SEAT_TEST_A1
- seatTypeId standard: SEAT_TYPE_STANDARD
- seatTypeId vip: SEAT_TYPE_VIP

Swagger auth:
1. POST /api/auth/login
2. Copy data.accessToken
3. Click Authorize
4. Paste token value into Bearer auth box

============================================================
ROOMS
============================================================

GET /api/admin/rooms
Role: Manager or Admin
Authorize with: manager@test.com token or admin@test.com token
Body: none

GET /api/admin/rooms/{roomId}
Example: GET /api/admin/rooms/ROOM_TEST_A
Role: Manager or Admin
Body: none

POST /api/admin/cinemas/{cinemaId}/rooms
Example: POST /api/admin/cinemas/CIN_TEST_ALL_ROLES/rooms
Role: Manager or Admin
Body:
{
  "roomName": "Room Test B",
  "capacity": 20,
  "roomStatus": "ACTIVE"
}

PUT /api/admin/rooms/{roomId}
Example: PUT /api/admin/rooms/ROOM_TEST_A
Role: Manager or Admin
Body:
{
  "roomName": "Room Test A Updated",
  "capacity": 12,
  "roomStatus": "ACTIVE"
}

DELETE /api/admin/rooms/{roomId}
Example: DELETE /api/admin/rooms/ROOM_TEST_A
Role: Manager or Admin
Body: none
Warning: Current service soft-deletes room by setting roomStatus = INACTIVE.

POST /api/admin/{roomId}/generate-seats
Example: POST /api/admin/ROOM_TEST_B_ID/generate-seats
Role: Manager or Admin
Note: Use a newly created room that has no seats yet. Existing ROOM_TEST_A already has seats.
Body:
{
  "rows": 4,
  "columns": 5,
  "seatTypeId": "SEAT_TYPE_STANDARD"
}

============================================================
SEATS
============================================================

POST /api/seats
Role: Manager only
Authorize with: manager@test.com token
Note: This creates a pending seat change request. Admin must approve it.
Body:
{
  "roomId": "ROOM_TEST_A",
  "rowLabel": "D",
  "seatNumber": 1,
  "seatTypeId": "SEAT_TYPE_STANDARD"
}

PUT /api/seats/{seatId}
Example: PUT /api/seats/SEAT_TEST_A1
Role: Manager only
Note: This creates a pending seat update request. Admin must approve it.
Body:
{
  "rowLabel": "A",
  "seatNumber": 1,
  "seatTypeId": "SEAT_TYPE_VIP"
}

DELETE /api/seats/{seatId}
Example: DELETE /api/seats/SEAT_TEST_A2
Role: Manager only
Note: This creates a pending seat delete request. Admin must approve it.
Body: none

GET /api/seats/room/{roomId}
Example: GET /api/seats/room/ROOM_TEST_A
Role: Manager or Admin
Body: none

GET /api/seats/requests/pending
Role: Admin only
Authorize with: admin@test.com token
Body: none

POST /api/seats/requests/{requestId}/approve
Role: Admin only
Authorize with: admin@test.com token
Note: Get requestId from GET /api/seats/requests/pending.
Example: POST /api/seats/requests/PASTE_REQUEST_ID/approve
Body: none

POST /api/seats/requests/{requestId}/reject
Role: Admin only
Authorize with: admin@test.com token
Note: Get requestId from GET /api/seats/requests/pending.
Example: POST /api/seats/requests/PASTE_REQUEST_ID/reject
Body:
{
  "reason": "Seat layout needs review."
}

POST /api/seats/lock
Role: Customer only
Authorize with: customer@test.com token
Body:
{
  "showtimeId": "SHOW_TEST_20260610_1900",
  "seatId": "SEAT_TEST_A1"
}

GET /api/seats/showtimes/{showtimeId}/map
Example: GET /api/seats/showtimes/SHOW_TEST_20260610_1900/map
Role: Customer, Manager, or Admin
Body: none
